# README

Rights Forward is a work platform and search engine for trainers in the non-profit field. 

## Code

Ruby on Rails, HTML, CSS, Javascript, jQuery

## Configuration & Dependencies
Clone the repo
Run:
``bundle install``
``rails db:create``
``rails db:seed``
``rails db:migrate``
``rails s``
Server up and running!

Rails 4
