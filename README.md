# README

Rights Forward is an employment platform and search engine for trainers in the non-profit field. 

## Code

Ruby on Rails, HTML, CSS, Javascript, jQuery

## Configuration & Dependencies
Clone the repo <br>
Run:<br>
``bundle install``<br>
``rails db:create``<br>
``rails db:seed``<br>
``rails db:migrate``<br>
``rails s``<br>
Server up and running!

## Ruby Version
Rails 4

## How it works
Users can sign up and create a profile
Users can add skills and expertise 
Users or visitors can search and view the profile of other users
